# 邮件服务 - Resend

ShipAny 支持使用 Resend 发送邮件，可用于发送邮件验证码、发送支付通知、发送营销邮件等场景。

## Resend 设置

1. 在 Resend 网站注册账号并登录后台
2. 在 Resend 后台添加域名，设置 DNS 记录，验证域名
3. 在 Resend 后台创建 API 密钥，并复制 API 密钥
4. 在 ShipAny 项目配置文件中，填写 Resend 相关的配置信息。
